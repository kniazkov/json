class JsonNumber extends JsonElement :

Represents a JSON element containing number.

A JSON number has the following method, in addition to inherited methods:

JsonNumber(double value)

Constructor. Creates an element that does not have a parent.

The overridden isNumber() method for this class will return true. You can also check whether a number is an integer (32-bit) or a long integer (64-bit) using the isInteger() and isLongInteger() methods.
Using the getIntValue(), getLongValue(), getDoubleValue() methods you can get the value of the element.
